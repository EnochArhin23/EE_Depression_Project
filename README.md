# Hippocampal Predictive Learning under Environmental Enrichment and Depression

## Overview
This project is a computational neuroscience model investigating how environmental enrichment influences hippocampal predictive learning under depression-like conditions using place cell representations and BTSP-inspired synaptic plasticity.

## Key Idea
We simulate how spatial and goal-directed learning emerges from hippocampal representations and how this process is affected by environmental complexity and reduced plasticity (depression-like state).

## Methods
- RatInABox simulation environment
- Hippocampal place cell encoding
- BTSP-inspired predictive learning rule
- Goal-directed reward prediction
- Comparison of enriched vs standard environments

## Key Findings
- Environmental enrichment improves spatial coding quality
- Enrichment enhances predictive learning accuracy
- Goal-directed prediction is impaired under low plasticity conditions
- Multi-timescale integration improves learning stability

## Biological Relevance
The model relates to hippocampal predictive coding, dendritic computation, and environmental modulation of cognitive function in depression.

## Dependencies
- Python 3.10+
- RatInABox
- NumPy
- Matplotlib
